# Stellar Burgers Unit Tests

## Описание проекта

Проект содержит юнит-тесты для программы заказа бургеров в сервисе **Stellar Burgers**.
Цель работы — протестировать класс **Burger** и обеспечить полное покрытие кода тестами.

## Проверяемый функционал

* добавление ингредиентов в бургер
* удаление ингредиентов
* перемещение ингредиентов
* расчёт стоимости бургера
* получение названия бургера

## Технологии

* Java 11
* Maven
* JUnit 4
* Mockito
* JaCoCo

## Особенности

* используются **моки и стабы Mockito**
* применена **параметризация тестов**
* покрытие кода с помощью **JaCoCo — 100%**
