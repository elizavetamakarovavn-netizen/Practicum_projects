## Описание проекта

Проект посвящён **ручному тестированию веб- и мобильного приложения**.
В рамках работы были изучены требования и макеты интерфейсов, после чего разработана тестовая документация и проведено тестирование функциональности приложения.

## Выполненные задачи

**Тестирование веб-приложения**

* составлен **чек-лист** для экрана «Статус заказа»
* разработаны **проверки валидации полей** для экрана «Сделать заказ»
* проведено тестирование функциональности по требованиям и макетам из **Figma**
* оформлены **баг-репорты** для найденных дефектов

**Тестирование мобильного приложения**

* разработаны **тест-кейсы** для основной функциональности
* добавлены проверки **верстки интерфейса по макетам**

## Результаты

* создана тестовая документация: чек-листы, таблицы валидации и тест-кейсы
* проведено функциональное тестирование
* задокументированы найденные дефекты

## Инструменты

## English

## Project Description

This project is dedicated to **manual testing of a web and mobile application**.
During the project, the requirements and interface mockups were analyzed, after which test documentation was created and functional testing of the application was performed.

## Tasks Performed

### Web Application Testing

* created a **checklist** for the **Order Status** screen
* designed **field validation checks** for the **Place Order** screen
* performed functional testing based on **requirements and Figma mockups**
* created **bug reports** for identified defects

### Mobile Application Testing

* developed **test cases** for the main functionality
* added **UI layout checks** based on design mockups

## Results

* created test documentation: **checklists, validation tables and test cases**
* performed **functional testing**
* documented the **identified defects**

## Tools

* Google Sheets
* Figma
* test documentation (**checklists, test cases, bug reports**)
