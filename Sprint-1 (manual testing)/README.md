# Mesto Service Regression Testing

## Russian

## Описание проекта
Проведено регрессионное тестирование веб-сервиса Mesto по готовым тест-кейсам.  
Выполнено тестирование на тестовом стенде, проставлены статусы проверок, заведены баг-репорты.  
После исправлений выполнен ретест багов для подтверждения корректности правок.

## English

## Project Description

Regression testing of the **Mesto web service** was performed based on prepared test cases.
Testing was carried out on a **test environment**, where execution results were recorded and **bug reports were created for identified defects**.

After the issues were fixed, **bug retesting** was performed to confirm that the fixes worked correctly.
