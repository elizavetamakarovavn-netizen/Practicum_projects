## Russian

## Описание проекта
Проведён тест-анализ логики полей формы сервиса Яндекс.Маршруты.  
Выделены классы эквивалентности и граничные значения, составлены тест-кейсы, выполнено тестирование и заведены баг-репорты.

## English

## Project Description

Test analysis of the **field logic in the Yandex Routes service form** was performed.
Equivalence classes and boundary values were identified, **test cases were designed**, testing was conducted, and **bug reports were created for detected issues**.
