# QA Java Unit Tests

## Описание проекта

Проект посвящён тестированию программы, используемой для исследования семейства кошачьих.
Цель работы — написать юнит-тесты для классов **Feline**, **Cat** и **Lion**, обеспечив полное покрытие кода.

## Проверяемый функционал

- Тестирование методов классов **Feline**, **Cat** и **Lion**
- Проверка всех веток условий
- Использование **параметризованных тестов**
- Использование **Mockito** для создания моков
- Реализация **Dependency Injection**, чтобы класс `Lion` не зависел напрямую от `Feline`

## Технологии

- Java 11
- Maven
- JUnit 4
- Mockito
- Jacoco

## Особенности

- Для каждого метода написан отдельный тест
- Все ветки условий покрыты тестами
- Покрытие кода с помощью **Jacoco — 100%**
