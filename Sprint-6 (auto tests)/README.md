## Russian

## Описание проекта

Проект посвящён тестированию программы, используемой для исследования семейства кошачьих.
Цель работы — написать юнит-тесты для классов **Feline**, **Cat** и **Lion**, обеспечив полное покрытие кода.

## Проверяемый функционал

- Тестирование методов классов **Feline**, **Cat** и **Lion**
- Проверка всех веток условий
- Использование **параметризованных тестов**
- Использование **Mockito** для создания моков
- Реализация **Dependency Injection**, чтобы класс `Lion` не зависел напрямую от `Feline`

## Технологии

- Java 11
- Maven
- JUnit 4
- Mockito
- Jacoco

## Особенности

- Для каждого метода написан отдельный тест
- Все ветки условий покрыты тестами
- Покрытие кода с помощью **Jacoco — 100%**

## English

## Project Description

This project focuses on testing a program used for **researching the feline family**.
The goal of the project is to write **unit tests for the classes `Feline`, `Cat`, and `Lion`** and achieve **full code coverage**.

## Tested Functionality

* testing methods of the classes **Feline, Cat, and Lion**
* verification of **all conditional branches**
* use of **parameterized tests**
* use of **Mockito** to create mocks
* implementation of **Dependency Injection** so that the `Lion` class does not depend directly on `Feline`

## Technologies

* Java 11
* Maven
* JUnit 4
* Mockito
* JaCoCo

## Features

* a **separate test for each method**
* **all conditional branches are covered by tests**
* **100% code coverage with JaCoCo**
