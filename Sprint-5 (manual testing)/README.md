# Тестирование API сервиса Яндекс.Прилавок

## Описание проекта
Проанализированы требования к новой функциональности бэкенда сервиса Яндекс.Прилавок.  
На основе документации API (Apidoc) спроектированы тесты в виде чек-листа.  
Проведено тестирование API через Postman, по найденным дефектам заведены баг-репорты.
