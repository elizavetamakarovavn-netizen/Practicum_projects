## Russian

## Описание проекта
Проанализированы требования к новой функциональности бэкенда сервиса Яндекс.Прилавок.  
На основе документации API (Apidoc) спроектированы тесты в виде чек-листа.  
Проведено тестирование API через Postman, по найденным дефектам заведены баг-репорты.

## English

## Project Description

The requirements for the new backend functionality of the **Yandex Lavka service** were analyzed.

Based on the **API documentation (Apidoc)**, tests were designed in the form of a **checklist**.
API testing was performed using **Postman**, and **bug reports were created for the identified defects**.
