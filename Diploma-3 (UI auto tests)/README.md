# Diplom_3 — UI-тесты (Java + Selenium + Allure)
Проект содержит автоматизированные UI-тесты для веб-приложения.
Тесты реализованы на Java 11, с использованием Selenium WebDriver, JUnit 4, WebDriverManager и Allure для генерации отчетов.

Тесты построены на Page Object и разделены по функциональным модулям:
 - регистрация;
 - авторизация;
 - проверка конструктора;
 - взаимодействие с вкладками интерфейса.

## Используемые технологии

| Компонент            | Технология / Версия |
|----------------------|----------------------|
| Язык программирования | Java 11 |
| Система сборки | Maven 3.9+ |
| Фреймворк для UI-тестов | Selenium WebDriver 4.35.0 |
| Тестовый фреймворк | JUnit 4.13.2 |
| Управление драйверами | WebDriverManager 5.5.0 |
| Отчётность | Allure 2.15.0 |
| Поддержка аспектов | AspectJ Weaver 1.9.7 |

**Запуск всех тестов:** mvn clean test

**Запуск тестов в Яндекс.Браузере:** mvn -Dtest=<имя теста> -Dbrowser=yandex test

**Запуск тестов в Google Chrome:** mvn -Dtest=<имя теста> -Dbrowser=chrome test

**Allure-отчет:** allure serve target/allure-results
