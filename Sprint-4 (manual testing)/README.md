# Yandex Metro Mobile App Regression Testing

## Russian

## Описание проекта
После рефакторинга Android-приложения Яндекс Метро выполнено тестирование изменённых частей продукта.  
Составлен чек-лист функционального и регрессионного тестирования с учётом особенностей мобильного приложения.  
Проведено тестирование и заведены баг-репорты.

## English

## Project Description

After refactoring the **Yandex Metro Android application**, testing of the modified parts of the product was performed.

A **checklist for functional and regression testing** was created, taking into account the specifics of mobile applications.
Testing was carried out and **bug reports were created for the identified defects**.
